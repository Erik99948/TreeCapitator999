🌲 TreeCapitator999

This plugin makes trees fall instantly when you break a log with an axe.

📦 Features

  • Break one log, and the whole tree falls
  • Supports all vanilla trees
  • Only works with axes

🧪 Commands

  • /treecap toggle – Enable or disable for yourself

⚙️ Permissions

  • treecap.use – Allows using the plugin
